# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/eae5310b-c958-4cb4-8e56-b8a7550c0074

## Comportamento

- As notificações não aparecem mais automaticamente.
- Para gerar um novo card de notificação, dê um **duplo clique no cabeçalho** (área central onde ficava o saldo).
- Um contador oculto aleatório de **5 a 10 segundos** é iniciado e, ao final, um novo card surge.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
